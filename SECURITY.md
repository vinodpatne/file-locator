# Security Policy

## Supported Versions

File Locator currently supports the following versions with security updates:

| Version   | Supported          |
| --------- | ------------------ |
| 1.0.x     | :white_check_mark: |
| < 1.0     | :x:                |

## Reporting a Vulnerability

We take the security of File Locator seriously. If you discover a security vulnerability, please report it by opening a new issue in the GitHub repository.

Please include the following details in your report:
* A description of the vulnerability and its potential impact.
* Steps to reproduce the issue.
* Any relevant environment details (e.g., OS version, Java version).

We aim to review and address all reported security vulnerabilities promptly. Please do not disclose the vulnerability publicly until a fix has been provided.
